# 季节循环系统

> 状态：Draft
> 层级：Foundation
> 优先级：MVP

## 概述

管理春夏秋冬四季的循环流转，每个季节持续 3-12 个随机回合。季节与五行属性共同决定气→分数的转化效率，是游戏经济系统的核心驱动。

## 玩家体验

玩家感受到季节的呼吸感——春天来了，木牌开始涨；夏天来了，火牌接力。好的玩家能预判季节切换的节奏，提前布局，在效率最高的窗口集中操作。

## 详细规则

### 杠杆乘数表（表B — 季节递增）

杠杆倍数随当前季节内已进行的回合数递增，创造择时博弈空间：

| 季节内回合 | 杠杆倍数 | 说明 |
|-----------|---------|------|
| 第 1-3 回合 | 1.0x | 无杠杆，安全期 |
| 第 4-6 回合 | 1.5x | 轻度杠杆 |
| 第 7-9 回合 | 2.0x | 中度杠杆 |
| 第 10-11 回合 | 2.5x | 较高杠杆 |
| 第 12 回合 | 3.0x | 最高杠杆（季节末尾，换季风险最大） |

> 设计意图：季节早期杠杆低但安全，晚期杠杆高但换季风险大。追涨杀跌型玩家在早期低杠杆买入赚快钱，长期持有者不会使用杠杆（持仓气耗 × 倍数太高）。

### 四季与五行的圆盘模型

```
        春(木)
       ↗    ↘
  冬(水)      夏(火)
       ↖    ↙
        秋(金)

  土 = 圆心，不受季节位置影响
  木火 = 一组（顺时针相邻）
  金水 = 一组（顺时针相邻）
  木↔金 = 对立，火↔水 = 对立
```

### 季节与五行的盈亏关系

同组内永远正收益，跨组才有亏损：

| | 春(木) | 夏(火) | 秋(金) | 冬(水) |
|---|--------|--------|--------|--------|
| **木** | 当季 ⭐⭐⭐ | 同组 ⭐⭐ | 对立 ❌❌ | 跨组 ⚠️ |
| **火** | 同组 ⭐⭐ | 当季 ⭐⭐⭐ | 跨组 ⚠️ | 对立 ❌❌ |
| **金** | 对立 ❌❌ | 跨组 ⚠️ | 当季 ⭐⭐⭐ | 同组 ⭐⭐ |
| **水** | 跨组 ⚠️ | 对立 ❌❌ | 同组 ⭐⭐ | 当季 ⭐⭐⭐ |
| **土** | 稳定 🟡 | 稳定 🟡 | 稳定 🟡 | 稳定 🟡 |

### 季节流转

- 顺序固定：春 → 夏 → 秋 → 冬 → 春 → ...
- 每个季节持续 **3-12 回合**，游戏开始时随机生成每个季节的长度
- 季节进度对玩家**不完全可见**（只显示"当前第几回合"，**隐藏本季总回合数**，保持对换季时机的未知感）
- 季节切换时，UI 明确提示玩家

### 核心框架：气→分数转化效率

所有得分操作的本质都是**气转化成分数**，转化效率由季节×五行决定：

```
效率 = 季节五行倍率 × (1 + 扩展修正)
```

**两种转化方式：**

| 方式 | 气消耗 | 分数产出 | 特点 |
|------|--------|---------|------|
| 持仓收益 | 买入时消耗一次 | 每回合持续结算 | 长线，省气，靠时间积累 |
| 卖出收益 | 买入+卖出消耗两次 | 卖出时一次性结算 | 短线，费气，集中爆发 |

### 整局资源模型（待跑测调参）

```
一局 60 回合
气上限 80
气回复速度：每回合 QR(7)，等待额外 WR(10)

卖出消耗：固定 3 气
持仓气耗：max(0.5, 1.5 + 0.4 × 评分) × 杠杆倍数

→ 整局大约能操作 N 次
→ 每次操作平均转化 M 分
→ 目标总分区间：[待定]
```

## 公式

### 持仓盈亏（每回合结算）

```
每回合得分 = 持有牌的盈亏倍率 × 基础分值
```

| 位置关系 | 持仓倍率 | 示例 |
|---------|---------|------|
| 当季 | +HOLD_MAX | 木牌在春 |
| 同组 | +HOLD_MID | 木牌在夏 |
| 跨组 | -HOLD_MINOR | 木牌在冬 |
| 对立 | -HOLD_MAJOR | 木牌在秋 |
| 土（任意季节） | +HOLD_STABLE | 稳定正收益 |

### 卖出盈亏（一次性结算）

```
卖出得分 = 卖出倍率 × 基础分值
```

| 位置关系 | 卖出倍率 |
|---------|---------|
| 当季 | +SELL_MAX |
| 同组 | +SELL_MID |
| 跨组 | -SELL_MINOR |
| 对立 | -SELL_MAJOR |
| 土（任意季节） | +SELL_STABLE |

> **设计约束**：卖出倍率 > 持仓单回合倍率（约 3-5 倍），确保短线操作有存在价值。

### 气消耗

```
买入消耗 = 基础消耗 × 季节系数
卖出消耗 = 基础消耗 × 卖出系数
```

| 情况 | 消耗系数 |
|------|---------|
| 买入当季属性牌 | COST_HIGH（最贵） |
| 买入同组属性牌 | COST_MID |
| 买入跨组属性牌 | COST_LOW |
| 买入对立属性牌 | COST_LOWEST |
| 买入土牌 | COST_BASE |
| 卖出 | COST_SELL（固定） |

> 逻辑：当季牌最值钱所以最贵，非当季牌便宜——在便宜时买入，贵时卖出，就是核心博弈。

## 边界情况

| 场景 | 处理方式 |
|------|----------|
| 一个季节只有 3 回合就结束了 | 玩家来不及布局，体现随机性风险 |
| 一个季节持续 12 回合 | 长线策略收益最大化，但要注意别错过下季 |
| 连续短季节（3+3+3+3） | 短线策略优势，持仓收益累计少 |
| 连续长季节（12+12+12+12） | 长线策略优势，季节可预测性强 |
| 游戏结束时恰好是某季中途 | 持仓牌正常结算最后一回合 |
| 60 回合不在季节边界结束 | 正常，季节在第 60 回合自然中断 |

## 验收标准

- [ ] 春→夏→秋→冬循环正确，顺序不乱
- [ ] 每季 3-12 回合随机生成，游戏开始时确定
- [ ] 季节进度可见（仅显示当前回合，隐去总回合）
- [ ] 季节切换有明确提示
- [ ] 盈亏倍率表正确对应五行×季节关系
- [ ] 同组内永远正收益，跨组才有亏损
- [ ] 土牌在任意季节稳定正收益
- [ ] 所有倍率参数可通过配置文件调整，无需改代码
