# 游戏概念：甲子纪（Jiazi Chronicle）

*创建时间：2026-04-25*
*状态：Draft*

## 电梯演讲

> 这是一款以六十甲子为主题的回合制策略卡牌经营游戏，玩家在春夏秋冬的随机轮回中，通过吞吐干支卡牌来积累分数——时机就是一切。

## 核心身份

| 方面 | 细节 |
|------|------|
| **类型** | 策略 / 卡牌 / 经营 / Roguelike |
| **平台** | Web（浏览器即玩）→ 后期可移植 PC |
| **玩家人数** | 单人（后期考虑双人） |
| **单局时长** | 约 20-40 分钟（60 回合） |
| **参考游戏** | 杀戮尖塔（卡牌决策）、循环英雄（季节循环 + 经营）、小丑牌（特殊规则构建） |

## 核心幻想

玩家是一位通晓天机五行运转的命师，在甲子轮回中洞察季节更替的规律，低吸高抛干支之气，在六十回合的有限时间窗口内积累最大的天命分数。每一张牌都是一次判断——**什么时候进场，什么时候离场，什么时候按兵不动**。

## 独特 Hook

> 像金融交易游戏一样紧张，但底层逻辑是五行生克与季节轮回——每一张干支牌都是一支"股票"，而"气"是你唯一的交易筹码。

## MDA 分析

### 目标美学（优先级排序）

| 美学 | 优先级 | 交付方式 |
|------|--------|----------|
| Challenge（挑战） | ★★★★★ | 不确定的季节长度 + 卡牌资源有限 + 气的博弈 |
| Submission（沉浸） | ★★★★☆ | 节奏感：等待→决策→结算，循环往复 |
| Discovery（发现） | ★★★☆☆ | 不同甲子组合的效果 + 季节长度的随机性 |
| Narrative（叙事） | ★★☆☆☆ | 干支本身的五行叙事（甲子乙丑各自的故事感） |

### 核心机制（3-5 个）

1. **季节循环系统** — 春夏秋冬固定顺序，每季 3-12 回合随机，驱动全局涨跌
2. **气资源系统** — 交易消耗气，不操作回复气，制造"等 vs 动"的核心博弈
3. **干支卡牌系统** — 60 甲子牌，天干地支各自带五行，决定季节盈亏（12 张特殊牌留到 Alpha）
4. **买卖决策系统** — 每回合只能买/卖/等三选一，强制玩家做取舍
5. **杠杆系统** — 季节递增倍数（1x→3x），追涨杀跌的天花板工具，长期持有者不会用

## 核心循环

### 瞬间到瞬间（30 秒）
- 看公共牌池 2 张牌 → 评估当前季节 → 决定买/卖/等

### 短期（3-12 回合）
- 一个季节内：判断当季什么属性赚钱 → 吸纳对应卡牌 → 在季节结束前考虑卖出

### 中期（一轮四季）
- 经历完整的涨跌周期 → 利用跨季节的买卖差价 → 积累分数优势

### 长期（60 回合全局）
- 管理气资源总量 → 在有限的 60 张牌中挑最有价值的 → 最终总分最大化

## 游戏支柱（4 个不可妥协的设计原则）

1. **时机至上**：玩家的核心决策永远是"什么时候做"而不是"做什么"——买同一张牌，在不同时机结果天差地别。设计测试：任何改动是否削弱了时机判断的重要性？

2. **信息不完全**：季节长度随机，公共牌池有不确定性——玩家永远无法完美预测，只能概率判断。设计测试：是否存在一种"必胜策略"让随机性变得无关紧要？

3. **简单规则，深度博弈**：规则一句话能说清（买低卖高，看季节），但实际决策有层次。设计测试：新手 5 分钟能上手，但高手和菜鸟的分数差距明显吗？

4. **每一个决策都有深刻的机会成本**：买卖会消耗气、改变仓位；而"等待"虽然能回气，代价却是眼睁睁看公共牌池的牌流走，同时也消耗了单局 60 回合中宝贵的 1 轮倒计时。由于"持仓收益"每回合是自动结算的，所以只要你不买不卖，就是在被动靠持仓盈亏——即便你捏着好牌在"躺赚"，选择等待也意味着你放弃了通过倒卖套短线快钱的机会。设计测试：玩家是否在任何时候都能感受到不管选什么，都有患得患失的重量？

### 反支柱（这个游戏不是什么）

- **不是数值养成游戏**：没有等级、没有装备强化，核心是单局决策
- **不是收集游戏**：卡牌是交易品不是收藏品，来了又走
- **不是运气游戏**：随机性提供多样性，但好玩家平均分数一定更高

## 五行与季节体系

| 五行 | 对应季节 | 卡牌特性 |
|------|----------|----------|
| 木 | 春 | 当季正收益高，其他季节衰减 |
| 火 | 夏 | 当季正收益高，其他季节衰减 |
| 金 | 秋 | 当季正收益高，其他季节衰减 |
| 水 | 冬 | 当季正收益高，其他季节衰减 |
| 土 | 无对应 | 稳定防守型，所有季节小幅正收益，无爆发 |

## 核心规则摘要

### 回合流程（每回合）

```
1. 游戏结束检查 → 如果当前回合 > 60，进入游戏结束
2. 季节检查 → 季节结束则切换到下一季
3. 持仓结算 + 持仓气耗扣除 → 根据持有卡牌 × 当前季节算本回合盈亏
4. 爆仓检查 → 气归零时随机强平杠杆牌
5. 刷牌 → 公共牌池展示 2 张牌（从剩余牌堆抽取）
6. 气回复 → 每回合自然回复 7 气；如果上回合选择"等待"，额外回复 10 气
7. 玩家决策 → 三选一：
   - 买入（消耗气，选 1 张纳入手牌，另一张回牌堆随机位置；可选杠杆）
   - 卖出（消耗 3 气，从手牌卖出 1 张，获得一次性盈亏分数，即时回复 8 气）
   - 等待（不消耗气，公共牌回牌堆，下回合额外回气）
8. 推进回合 → 当前回合 +1
```

### 手牌与牌库

| 项目 | 规则 |
|------|------|
| 手牌上限 | 3 个格子 |
| 总卡牌 | 60 甲子牌（MVP）；12 张特殊牌留到 Alpha |
| 牌池展示 | 每回合 2 选 1 |
| 牌的去向 | 买入后从牌堆永久移除；未选的回牌堆随机位置 |
| 买入限制 | 格子满则不能买入 |
| 杠杆 | 买入时可选，季节递增倍数（1x→1.5x→2x→2.5x→3x） |

### 气资源

| 项目 | 规则 |
|------|------|
| 气上限 | 80 |
| 初始气 | 50 |
| 回复条件 | 每回合自然回复 7 气；选择"等待"时额外回复 10 气 |
| 消耗条件 | 执行买入或卖出时消耗；卖出即时回复 8 气 |
| 消耗差异 | 买入成本 = 12 × (1 + 0.05 × 评分)；持仓气耗 = max(0.5, 1.5 + 0.4 × 评分) × 杠杆 |

### 特殊牌：空亡（Alpha 阶段实现）

> MVP 阶段未实现特殊牌。以下为 Alpha 阶段设计：

| 效果 | 规则 |
|------|------|
| 触发条件 | 从公共牌池翻出 |
| 玩家操作 | 没收本回合操作权（强制跳过操作阶段），但依然会自动结算本回合的持仓收益 |
| 季节影响 | 季节进度瞬间加速，突然向前推进 3~6 回合（极大增加当前季节提前结束的风险） |
| 战略意义 | 因为持仓牌仍会结算分数，空亡牌是福是祸完全取决于当前持仓质量，也可能帮你更快熬过坏季节 |

## MVP 定义

**核心假设**：五行季节涨跌 + 气资源博弈的决策循环是否有趣？

**MVP 必须包含**：
- [ ] 60 甲子牌的基础属性（天干、地支、五行）
- [ ] 季节循环系统（春→夏→秋→冬，3-12 回合随机）
- [ ] 公共牌池 2 选 1
- [ ] 气资源系统（80 上限，50 初始，等待回复，买卖消耗）
- [ ] 3 格手牌 + 买卖操作
- [ ] 每回合盈亏结算 + 总分累计
- [ ] 60 回合结束 → 显示最终得分

**MVP 不包含**：
- 12 张特殊牌（空亡牌留到 Alpha）
- 双人模式
- 美术打磨（MVP 用简单色块/文字）
- 音效
- 甲子组合联动加成
- AI 对手

## 风险

| 风险 | 等级 | 应对策略 |
|------|------|----------|
| 季节长度随机导致极端局（一直春天或一直冬天） | 中 | 3-12 范围已经有限制，测试后调整 |
| 气的回复/消耗比例不平衡导致玩家被迫一直等 | 高 | MVP 重点调参目标 |
| 玩法太单调（买低卖高没有更多层次） | 中 | 后续通过甲子组合加成增加深度 |
| 特殊牌空亡的突然加速可能让玩家感到突兀 | 低 | Alpha 阶段验证，需用特效清晰表达"季节被推着走"的概念 |
