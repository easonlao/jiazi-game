# 杠杆系统

> 状态：Draft
> 层级：Core
> 优先级：MVP

## 概述

玩家在买入卡牌时可以选择使用杠杆，放大该牌的持仓收益和卖出收益。杠杆倍数由当前季节内已进行的回合数决定——季节越深，杠杆越高，但换季风险也越大。杠杆不是必选项，长期持有者不会使用它。

## 玩家体验

玩家在买入时看到一个"杠杆开关"和当前倍率显示。早期倍率低（1.0x~1.5x），晚期倍率高（2.5x~3.0x）。用杠杆赚了会很爽，但持仓气耗放大让人不敢久拿——追涨杀跌型玩家的完美工具。

## 详细规则

### 杠杆乘数表（表B — 季节递增）

| 季节内回合 | 杠杆倍数 | 说明 |
|-----------|---------|------|
| 第 1-3 回合 | 1.0x | 无杠杆，安全期 |
| 第 4-6 回合 | 1.5x | 轻度杠杆 |
| 第 7-9 回合 | 2.0x | 中度杠杆 |
| 第 10-11 回合 | 2.5x | 较高杠杆 |
| 第 12 回合 | 3.0x | 最高杠杆（季节末尾，换季风险最大）`

### 杠杆气消耗

使用杠杆买入时，气消耗分为两部分：

| 消耗类型 | 公式 | 说明 |
|---------|------|------|
| 买入额外消耗 | LQC = 10（固定） | 不随杠杆倍数变化，保证买得起 |
| 持仓气耗 | max(0.5, 1.5 + 0.4 × 当季评分) × 杠杆倍数 / 回合 / 张 | 评分越高气耗越大，高杠杆持有越久越贵 |

```
总买入气消耗 = CB(12) × (1 + 0.05 × 评分) + LQC(10)
每回合持仓气耗 = max(0.5, 1.5 + 0.4 × 当季评分) × 杠杆倍数 × 持仓杠杆牌数
```

### 杠杆对收益的影响

| 收益类型 | 公式 |
|---------|------|
| 持仓收益 | HOLD_BONUS(1.2) × 评分 × 杠杆倍数（每回合） |
| 卖出收益 | (SELL_BASE(8) + (卖出评分 - 买入评分) × SPREAD_MULTIPLIER(4)) × 杠杆倍数 |

### 爆仓机制

气归零时触发爆仓：

```
如果 气归零 且 持有杠杆牌:
    随机选择一张杠杆牌强制卖出（强平）
    强平卖出不消耗气
    强平卖出正常获得卖出收益（× 杠杆倍数）
    如果强平后仍持有杠杆牌且气仍为 0:
        重复强平直到无杠杆牌或气 > 0
```

### 杠杆标记

每张手牌需要记录：

```typescript
interface HandSlot {
  card: JiaziCard;
  buyScore: number;      // 买入时的评分
  leverage: number;       // 买入时的杠杆倍数（1.0 = 无杠杆）
  buyRound: number;       // 买入时的回合数
  holdEarnings: number;   // 累计持有收益（每回合利息之和）
}
```

## 公式

### 获取当前杠杆倍数

```typescript
function getCurrentLeverage(seasonRound: number): number {
  if (seasonRound <= 3) return 1.0;
  if (seasonRound <= 6) return 1.5;
  if (seasonRound <= 9) return 2.0;
  if (seasonRound <= 11) return 2.5;
  return 3.0;
}
```

### 杠杆买入气消耗

```typescript
function getLeverageBuyCost(cardScore: number): number {
  const baseCost = CB * (1 + 0.05 * cardScore);
  return baseCost + LQC; // LQC = 10
}
```

### 杠杆持仓气耗（与普通牌共用动态公式）

```typescript
function calcHoldQiCost(cardScore: number, leverage: number = 1.0): number {
  const base = Math.max(0.5, 1.5 + 0.4 * cardScore);
  return parseFloat((base * leverage).toFixed(1));
}
// 所有手牌的持仓气耗 = Σ calcHoldQiCost(当季评分(卡牌), 杠杆倍数)
```

## 边界情况

| 场景 | 处理方式 |
|------|----------|
| 季节第 1-3 回合使用杠杆 | 倍率 1.0x，等同于无杠杆，但仍然消耗 LQC（不划算） |
| 气刚好够杠杆买入 | 正常执行，买入后气接近 0 |
| 持有多张杠杆牌 | 每张独立计算持仓气耗，总持仓气耗 = Σ(各牌持仓气耗) |
| 气归零且持有多张杠杆牌 | 逐张随机强平，直到气 > 0 或无杠杆牌 |
| 杠杆牌被强平 | 正常计算卖出收益（× 杠杆倍数），强平不消耗气 |
| 季节切换后杠杆倍数重置 | 已持有牌的杠杆倍数不变（按买入时记录的倍数） |
| 全部手牌都是杠杆牌 | 持仓气耗极高，需要频繁操作回气或快速卖出 |

## 依赖关系

| 系统 | 方向 | 性质 |
|------|------|------|
| 季节循环 | 依赖 | 读取季节内回合数确定杠杆倍数 |
| 气资源 | 依赖 | 扣除杠杆买入消耗、持仓气耗 |
| 手牌管理 | 被依赖 | 在手牌上标记杠杆倍数 |
| 盈亏结算 | 被依赖 | 杠杆牌收益 × 杠杆倍数 |
| 回合流程 | 被依赖 | 每回合结算持仓气耗、检查爆仓 |

## 调参旋钮

| 参数 | 值 | 安全范围 | 影响 |
|------|-----|----------|------|
| LQC（杠杆额外消耗） | 10 | 5 ~ 15 | 杠杆买入的固定额外代价 |
| HOLD_QI_COST（持仓气耗公式） | max(0.5, 1.5+0.4×评分) | 1 ~ 5 | 基础持仓气耗（×杠杆倍数），评分越高气耗越大 |
| 杠杆倍数表 | 1.0→1.5→2.0→2.5→3.0 | — | 季节递增杠杆力度 |

## 验收标准

- [ ] 杠杆倍数按季节内回合数正确递增
- [ ] 季节切换后倍数重置（新季节第 1 回合 = 1.0x）
- [ ] 杠杆买入额外消耗 LQC，不乘杠杆倍数
- [ ] 持仓气耗 = max(0.5, 1.5+0.4×评分) × 杠杆倍数
- [ ] 杠杆牌持仓收益 × 杠杆倍数
- [ ] 杠杆牌卖出收益 × 杠杆倍数
- [ ] 气归零触发爆仓（随机强平杠杆牌）
- [ ] 爆仓强平不消耗气
- [ ] 已持有牌的杠杆倍数不受后续季节变化影响
- [ ] 杠杆倍数在 UI 上清晰展示
- [ ] 所有杠杆参数可通过配置文件调整
