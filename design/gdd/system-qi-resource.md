# 气资源系统

> 状态：Draft
> 层级：Foundation
> 优先级：MVP

## 概述

管理玩家的气资源——游戏中唯一的交易筹码。气的回复和消耗构成"等 vs 动"的核心博弈，是连接玩家决策与分数产出的桥梁。

## 玩家体验

玩家每回合都在感受气的压力：想操作但气不够，想等回气但又怕错过好牌。气是有限的，每一次花出去都要想清楚值不值。

## 详细规则

### 气的基本属性

| 属性 | 规则 |
|------|------|
| 上限 | 80 |
| 初始值 | 50 |
| 可否超出上限 | 不可以 |
| 最低值 | 0（归零时只能等待） |

### 气的回复

| 条件 | 规则 |
|------|------|
| 上回合选择"等待" | 回复 WR(10) + QR(7) = 17 点 |
| 上回合执行了买入/卖出 | 自然回复 QR(7) 点 |
| 回复可以超过上限 | 不可以，封顶 80 |

> 设计意图：鼓励玩家有时候按兵不动。连续操作会气竭，被迫等待。

### 气的消耗

| 操作 | 消耗逻辑 |
|------|---------|
| 买入（无杠杆） | 基础消耗 × (1 + 0.05 × 评分) |
| 买入（有杠杆） | 基础消耗 × (1 + 0.05 × 评分) + LQC（固定额外消耗，不随杠杆倍数变化） |
| 持仓气耗 | 基于卡牌强度动态计算 max(0.5, 1.5 + 0.4 × 当季评分) × 杠杆倍数 |
| 卖出 | 固定消耗 3 气 |
| 等待 | 0 消耗 |

#### 杠杆气消耗

使用杠杆买入时，气消耗分为两部分：

```
总买入气消耗 = CB(12) × (1 + 0.05 × 评分) + LQC(10)
持仓气耗/回合/张 = max(0.5, 1.5 + 0.4 × 当季评分) × 杠杆倍数
```

- **LQC（杠杆额外消耗）固定为 10 气**，不随杠杆倍数变化 → 保证任何杠杆级别都买得起
- **持仓气耗 × 杠杆倍数** → 高杠杆持有越久"利息"越贵，自然限制持有时间
- 气归零时触发**爆仓**：随机强平（强制卖出）一张杠杆持仓牌

#### 买入消耗

```
买入消耗 = CB × (1 + 0.05 × 买入时评分)
使用杠杆时：买入消耗 = CB × (1 + 0.05 × 买入时评分) + LQC(10)
```

- 评分高的牌（当季牌）买入更贵，评分低的牌（反季牌）更便宜
- 杠杆额外消耗 LQC 固定，不乘杠杆倍数

#### 气不足时的处理

| 场景 | 处理 |
|------|------|
| 气不够买入最便宜的牌 | 无法买入，只能等待或卖出 |
| 气不够卖出 | 无法卖出，只能等待或买入 |
| 气刚好够 | 正常执行，气归零 |
| 气归零且持有杠杆牌 | 触发爆仓：随机强平一张杠杆持仓牌 |

### 气在游戏经济中的位置

```
气(有限资源) ──消耗──→ 操作 ──产出──→ 分数
       ↑                        │
       └── 等待回复 ←── 季节×五行决定效率 ──┘

整局经济平衡：
  气回复速度 × 可等待回合数 = 总可用气
  总可用气 / 平均每次操作消耗 = 总可操作次数
  总可操作次数 × 平均每次转化分数 = 理论最大总分
```

## 公式

### 回复公式

```
每回合开始时：
    当前气 = min(80, 当前气 + QR(7))  // 自然回复

如果 上回合动作 == "等待":
    当前气 = min(80, 当前气 + WR(10))  // 额外等待回气
```

### 买入消耗公式

```
买入消耗 = CB × (1 + 0.05 × 买入时评分)

如果使用杠杆:
    买入消耗 += LQC(10)  // 固定额外消耗

如果 当前气 < 买入消耗:
    买入失败，提示气不足
否则:
    当前气 -= 买入消耗
    牌进入手牌（带杠杆倍数标记）
```

### 卖出消耗公式

```
卖出消耗 = 3（固定值）

如果 当前气 < 3:
    卖出失败，提示气不足
否则:
    当前气 -= 3
    手牌移除
    获得一次性卖出分数 = (SB(8) + (卖出评分 - 买入评分) × SM(4)) × 杠杆倍数
    立即回复 CS(8) 气（卖出操作即时生效，非下回合）
```

## 边界情况

| 场景 | 处理方式 |
|------|----------|
| 气为 0 | 只能等待，UI 上等待按钮高亮，买入/卖出置灰 |
| 气为 0 且持有杠杆牌 | 触发爆仓（随机强平），强平后气不变（强平不消耗气） |
| 气回复后刚好够买入 | 正常操作 |
| 连续多次等待气满 80 | 溢出部分浪费，气封顶 |
| 买入后气不足卖出 | 可以等待回气后再卖 |
| 游戏结束时气没用完 | 不转化成分数，浪费了 |

## 依赖关系

| 系统 | 方向 | 性质 |
|------|------|------|
| 干支卡牌数据 | 依赖 | 读取牌的主属性计算买入系数 |
| 季节循环 | 依赖 | 读取当前季节计算买入系数 |
| 公共牌池 | 被依赖 | 判断玩家是否有足够气买入 |
| 手牌管理 | 被依赖 | 判断玩家是否有足够气卖出 |
| UI 渲染 | 被依赖 | 展示气条、消耗预估 |

## 调参旋钮

> ⚠️ 以下数值为初始占位，需跑模拟测试后调整。

| 参数 | 占位值 | 安全范围 | 影响 |
|------|--------|----------|------|
| QR（每回合自然回复） | 7 | 5 ~ 10 | 基础回气速度 |
| WR（等待额外回复） | 10 | 5 ~ 15 | 等待的吸引力 |
| CB（买入基础消耗） | 12 | 8 ~ 20 | 所有买入的基数 |
| 卖出消耗 | 3 | 1 ~ 5 | 卖出成本 |
| CS（卖出回气） | 8 | 5 ~ 12 | 卖出的气补偿 |
| LQC（杠杆额外消耗） | 10 | 5 ~ 15 | 杠杆买入的固定额外代价 |
| HOLD_QI_COST（持仓气耗公式） | max(0.5, 1.5+0.4×评分) | 1 ~ 5 | 持有卡牌的每回合气成本（×杠杆倍数），评分越高气耗越大 |

### 调参参考指标

| 指标 | 目标 |
|------|------|
| 一局总可等待回合数 | 20-30 回合（玩家不能全程等） |
| 一局总可操作次数 | 15-25 次 |
| 全等待 vs 全操作的差距 | 操作得当的玩家得分应明显高于全等待者 |
| 气竭（被迫等待）频率 | 平均每 3-5 回合出现一次 |

## 验收标准

- [ ] 气上限 80，初始 50，不可超出
- [ ] 等待回合额外回复 WR(10) 气
- [ ] 每回合自然回复 QR(7) 气（不管上回合操作是什么）
- [ ] 卖出回复 CS(8) 气
- [ ] 买入消耗根据评分公式正确计算
- [ ] 杠杆买入额外消耗 LQC(10)，不乘杠杆倍数
- [ ] 持仓气耗 = HOLD_QI_COST × 杠杆倍数（每回合每张牌）
- [ ] 气归零时触发爆仓（随机强平杠杆牌）
- [ ] 气不足时对应操作不可执行
- [ ] 气为 0 时只能等待
- [ ] 所有消耗参数可通过配置文件调整
