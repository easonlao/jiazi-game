# Patch 0.0.1 — 首个可玩版本
*2026-05-04*

## 新内容

### 四季轮回
游戏现在包含春、夏、秋、冬四季，每个季节持续3-12个回合。季节会改变卡牌的价值：相生元素（如春天的木）会涨价，相克元素（如秋天的木）会跌价。

### 气资源系统
你拥有“气”作为主要资源，上限80，初始50。
- **买入卡牌**：消耗气（基础成本12 × (1 + 5% × 评分)），使用杠杆再加10
- **卖出卡牌**：消耗3气，但立即回复8气
- **持有卡牌**：每回合按卡牌评分和杠杆倍数扣除气
- **恢复**：每回合自动回复7气，如果上回合选择“等待”，额外回复10气

### 卡牌买卖系统
- **60张独特卡牌**，每张有自己的天干、地支、元素和阴阳
- **每回合展示2张公共牌**，你可以买入其中一张，或不买让它们回到牌堆随机位置
- **手牌最多3张**，持有的卡牌每回合自动产生分数（持仓收益）
- **卖出卡牌**可一次性获得分数（= 8 + (卖出评分 - 买入评分) × 4）× 杠杆

### 杠杆系统
随着季节推移，你可以选择为卡牌添加杠杆，放大中性收益和持仓成本：
- **多重叠数**：1.0x / 1.5x / 2.0x / 2.5x / 3.0x（赛季越深倍数越高）
- **额外成本**：使用杠杆需额外支付10气（不乘倍数）
- **平仓风险**：如果气归零，会随机强制卖出一张杠杆卡牌（没收操作方便）

### UI交互
- **显示分数**、**季节**、**回合**、**气条**
- **手牌区域**（3槽位）和**公共牌区域**（2槽位）
- **按钮**：买入、卖出、等待、杠杆开关
- 点击卡牌可选中，然后点击按钮执行操作

### 胜利条件
坚持60回合，积累最高分数即可胜利。
